## 📖 使用流程

1. **启动系统**
   ```bash
   ./start.sh
   ```

2. **访问界面**
   - 打开浏览器访问 http://localhost:8080

3. **添加目标**
   - 在"目标管理"中添加要监控的域名

4. **触发扫描**
   - 点击"扫描"按钮或等待自动扫描

5. **查看结果**
   - 在"子域名"查看发现的资产
   - 在"变更中心"查看资产变更

6. **配置通知**（可选）
   - 编辑 `backend/config.yaml`
   - 配置企业微信或钉钉 Webhook
   - 重启后端服务
